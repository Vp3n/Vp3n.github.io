Développement d’une application permettant d’évaluer les émissions de gaz à effet de serre mais aussi de montrer les efforts effectués pour les réduire au mieux.

  • Plateforme Java EE6 avec JSF2, Seam3, CDI/Weld, JPA/Hibernate
  • JEXEL
  • BIRT pour la génération de rapport détaillés
  • Web Service avec JAX-RS et JSON
  • Base de données SQLServer
  • jQuery
